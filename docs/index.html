Calibrate Pro

Professional Sensorless Display Calibration

Detect, characterize, and calibrate your displays with panel-accurate color
correction — no colorimeter required.

Download View on GitHub

Key Features

58-Panel Database

Characterized display profiles covering OLED, QD-OLED, IPS, and VA panels with predicted dE < 1.0 accuracy.

DDC/CI Hardware Control

Touch-free OSD configuration — picture mode, color preset, gamma, brightness, and RGB gains via DDC/CI.

3D LUT Generation

Residual correction 3D LUTs applied system-wide through DWM with VCGT gamma ramp fallback.

ICC Profile Export

ICC v4 profiles with per-channel TRC and measured primaries, installed automatically for system-wide color management.

Sensorless Mode

Full calibration without a colorimeter using panel-database characterization. Optional i1Display3 for measured accuracy.

CalibrationGuard Watchdog

15-second watchdog prevents Windows from resetting your calibration. Persists across reboots via startup integration.

How It Works

1

Detect

Identify connected displays and match to the panel database via EDID.

2

Characterize

Configure monitor OSD settings and adjust hardware controls via DDC/CI.

3

Calibrate

Generate a correction 3D LUT and ICC profile from panel or sensor data.

4

Verify

CalibrationGuard locks in your settings and protects against OS resets.

System Requirements

  • OS: Windows 10 / 11
  • Python: 3.10+
  • GUI: PyQt6
  • Sensor: Optional — i1Display3 family

Quick Start

Install

pip install ".[all]"

Launch GUI

calibrate-pro

CLI Usage

calibrate-pro auto          # Calibrate all displays (sensorless)
calibrate-pro detect        # Show connected displays + sensors
calibrate-pro ddc-info      # Show DDC/CI capabilities per monitor
